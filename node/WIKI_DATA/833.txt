= EP01. 막간 : 영웅에게로 =
 * 전설의 영웅
 애쉬와 더스트를 사냥하기 위해, 전설의 영웅인 알파퀸 서지수에게 협력을 구하기로 결정되었다. 해당 내역을 오세린도 전달받았는지 확인하고 앨리스에게 보고하자.
  * 임무: 앨리스와 대화 → 오세린과 대화 → 앨리스와 대화

 * 신서울, 강남으로
 신서울의 강남으로 지역 이동하고 해당 지역에서의 행동 요강을 전달받은 뒤, 김도윤에게 가벼운 충고를 전해주자.
  * 임무: 앨리스와 대화 → '''강남 GGV로 이동''' → 앨리스와 대화 → 김도윤과 대화

= EP02. 강남의 거리 =
 * 거리 구경
 강남의 명물, 포장마차 여우네에 들러 그 주인장과 대화한 후, 오세린과 감상을 나눠보자.
  * 임무: 김재리와 대화 → 채민우와 대화 → 오세린과 대화

 * 강남 재침공
 [강남역 인근]에서 차원종의 기척이 확인되었다. 출동하여 적을 처리하자.
  * 임무: 오세린과 대화 → 강남역 인근 [침투 (NORMAL) 난이도 이상] 1회 클리어 → 채민우와 대화

 * 대로변 확보
 출몰한 차원종이 민간인을 습격할 우려가 있다. [강남역 인근]의 차원종을 섬멸하자.
  * 임무: 채민우와 대화 → 강남역 인근 [섬멸 (HARD)] 2회 클리어 → 앨리스와 대화

== 완충장치 제작 ==
 [강남역 인근]에서 마나나폰의 잔해를 가져와 이빛나에게 전달하자.
 * 임무: 이빛나와 대화 → 강남역 인근 [섬멸 (HARD)] - 마나나폰 갑피 2개 수집 → 이빛나와 대화

 '강남 재침공' 퀘스트를 완료하면 퀘스트 출현.

= EP03. 알파퀸 탐색 =
 * 재의 소년
 가까운 곳에 이번 작전의 타겟, 더스트가 출현했을 가능성이 있다. [역삼 주택가]에서 알파퀸의 행방을 찾아 협조를 구하자.
  * 임무: 앨리스와 대화 → [싱글:1인] 역삼 주택가 [침투 (NORMAL)] 1회 클리어 → 앨리스와 대화

 * 영웅은 어디로
 홀연히 사라진 알파퀸, 서지수의 행방을 탐색하면서 [역삼 주택가]의 차원종을 섬멸하자.
  * 임무: 앨리스와 대화 → 역삼 주택가 [섬멸 (HARD)] 2회 클리어 → 채민우와 대화

== 공황 유발자들 ==
 [역삼 주택가]와 [시간의 광장]에 나타난 인형 타입 차원종을 처치하자.
 * 임무: 오세린과 대화 → 역삼 주택가 [섬멸 (HARD)] - 공포의 인형, 시간의 광장 [섬멸 (HARD)] - 분노의 인형 각각 2회 처치 → 오세린과 대화

 '재의 소년' 퀘스트를 완료하면 퀘스트 출현.   

= EP04. 쇼핑몰 수색 =
 * 폐허의 쇼핑몰
 목격자의 증언에 의하면, 알파퀸으로 추정되는 여성이 [시간의 광장]으로 향했다고 한다. 조사에 나서도록 하자.
  * 임무: 채민우와 대화 → 시간의 광장 [침투 (NORMAL) 난이도 이상] 1회 클리어 → 김도윤과 대화

 * 흔적 찾기
 좀처럼 발견되지 않는 알파퀸의 흔적을 찾아, [시간의 광장] 깊숙한 곳까지 조사해보도록 하자.
  * 임무: 김도윤과 대화 → 시간의 광장 [섬멸 (HARD)] 2회 클리어 → 오세린과 대화

= EP05. 민간인과 영웅 =
 * 먼지의 소녀
 [강남역 인근]의 차원종들이 광분하며 날뛰어대고 있다. 민간인의 구출을 위해 신속히 차원종들을 섬멸하자.
  * 임무: 오세린과 대화 → [싱글:1인] 강남역 인근 [정복 (V.HARD)] 1회 클리어 → 앨리스와 대화

 * 우선해야 할 것
 오퍼레이터인 앨리스의 허가가 떨어졌다. [강남역 인근]과 [역삼 주택가]의 차원종들을 격멸하여 민간인들을 보호하자.
  * 임무: 앨리스와 대화 → 강남역 인근 [정복 (V.HARD)], 역삼 주택가 [정복 (V.HARD)] 각각 1회 클리어 → 채민우와 대화

 * 쇼핑몰 수색 재개
 특경대의 협조 덕분에 다시 알파퀸 수색을 재개할 수 있게 되었다. [시간의 광장]을 빈틈없이 조사하자.
  * 임무: 채민우와 대화 → 시간의 광장 [정복 (V.HARD)] 2회 클리어 → 김재리와 대화

 * 클로저의 일
 김재리와 김도윤이 준비해준 무인 탐사 로봇이 알파퀸 수색을 대신해준다고 한다. 특경대원들을 도와 [역삼 주택가]의 민간인들을 보호하자.
  * 임무: 김재리와 대화 → 역삼 주택가 [정복 (V.HARD)] 2회 클리어 → 앨리스와 대화

== 웃는 남자 ==
 [강남역 인근]과 [역삼 주택가]에 나타난 인형 타입 차원종을 처치하자.
 * 임무: 채민우와 대화 → 강남역 인근 [정복 (V.HARD)] - 즐거운 인형 25개체, 역삼 주택가 [정복 (V.HARD)] - 기쁨의 인형 15개체 처치 → 채민우와 대화

 '먼지의 소녀' 퀘스트를 완료하면 퀘스트 출현.

== 인형의 심장 ==
 [시간의 광장]에서 퍼펫 마스터를 처치하고, 해당 차원종의 잔해를 오세린에게 전달하자.
 * 임무: 오세린과 대화 → 시간의 광장 [정복 (V.HARD)] - 퍼펫 마스터의 심장 2개 수집 → 오세린과 대화

 '우선해야 할 것' 퀘스트를 완료하면 퀘스트 출현.

= EP06. 여왕의 등장 =
 * 통쾌하게
 강남 수비 작전이 막바지에 이르렀다. [강남역 인근]의 적들을 시원하게 박살내버리자.
  * 임무: 앨리스와 대화 → 강남역 인근 [정복 (V.HARD)] 1회 클리어 → 채민우와 대화

 * 특경대의 보은
 특경대원들이 알파퀸 수색에 도움을 준다고 한다. 그들과 함께 [시간의 광장]에서 그녀의 흔적을 찾아보자.
  * 임무: 채민우와 대화 → [싱글:1인] 시간의 광장 [정복 (V.HARD)] 1회 클리어 → 김재리와 대화

 * 난 아니다
 [시간의 광장]에서 그녀의 소행으로 추정되는 흔적이 발견되었다. 원인을 규명하러 가보자.
  * 임무: 김재리와 대화 → [싱글:1인] 시간의 광장 [정복 (V.HARD)] 1회 클리어 → 앨리스와 대화

 * 한강으로
 알파퀸 서지수의 행방을 찾아 한강 상공으로 이동해야 한다. 앨리스, 김도윤과 대화하여 상황을 파악해보자.
  * 임무: 앨리스와 대화 → '''한강 상공으로 이동''' → 앨리스와 대화 → 김도윤과 대화

== 리시버 슬롯 해제 ==
 * 새로운 장비
 이빛나의 원활한 안부 통화를 도울 겸, 적 지휘관급이 지닌 [방해전파 발신기]와 다른 재료들을 모아 가져가서, 새로운 장비를 지급받아라.
  * 임무: 이빛나와 대화 → 한강 상공 [침투 (NORMAL) 난이도 이상] - 방해전파 발신기 3개, 반차원압 물질 50개, 확고한 차원의 균열 10개 수집 → 이빛나와 대화

 방해전파 발신기는 섬멸, 정복 난이도에서만 획득 가능하다.

 한강 상공에 도착하게 될 쯤에는 보통, 캐릭터의 레벨이 적정 레벨을 넘을 터라, 바로 이 퀘스트를 수락할 수 있다.      
 
= EP07. 강변의 작전 =
 * 한강 나들이 
 [한강 강변길] 일대를 정찰하며 차원종들을 물리치고 알파퀸의 흔적을 조사하자.
  * 임무: 김재리와 대화 → 한강 강변길 [침투 (NORMAL)] 1회 클리어 → 앨리스와 대화

 * 강변 수색
 추적용 로봇이 파괴되는 바람에 알파퀸의 흔적이 소실되었다. [한강 강변길]의 차원종들을 섬멸하며 행적을 탐색하자.
  * 임무: 앨리스와 대화 → 한강 강변길 [섬멸 (HARD)] 2회 클리어 → 채민우와 대화

== 숙주를 찾아서 ==
 [한강 강변길]에 나타난 패러사이트 타입의 잔해를 김도윤에게 전달하자.
 * 임무: 김도윤과 대화 → 한강 강변길 [섬멸 (HARD)] - 악취나는 살덩이 7개 수집 → 김도윤과 대화

 '한강 나들이' 퀘스트를 완료하면 퀘스트 출현.

= EP08. 한강의 다리 =
 * 대교의 클로저
 성수대교는 전략의 요충지로, 차원종들의 점거를 관망해서는 안 되는 지역이다. [성수대교]를 차원종에게서 방어하자.
  * 임무: 채민우와 대화 → 성수대교 [침투 (NORMAL) 난이도 이상] 1회 클리어 → 오세린과 대화

 * 뚫린 방어선
 차원종들은 계속해서 [성수대교]를 향해 진군하고 있다. 차원종들의 진격에 맞서 놈들을 섬멸하자.
  * 임무: 오세린과 대화 → 성수대교 [섬멸 (HARD)] 2회 클리어 → 김도윤과 대화

== 도심의 제초 ==
 [성수대교]와 [아파트 단지터]에 나타난 식물형 차원종들을 처치하자.
 * 임무: 정도연과 대화 → 성수대교 [섬멸 (HARD)] - 성장한 맨드란, 아파트 단지터 [섬멸 (HARD)] - 맨드란 각각 40개체 처치 → 정도연과 대화

 '대교의 클로저' 퀘스트를 완료하면 퀘스트 출현.

= EP09. 영웅의 옛 집 =
 * 무인 아파트 단지
 방어선을 뚫은 차원종들이 거주민이 없는 [아파트 단지터]를 향해 이동 중이다. 원인을 규명하기 위해 해당 지역을 조사하자.
  * 임무: 김도윤과 대화 → 아파트 단지터 [침투 (NORMAL) 난이도 이상] 1회 클리어 → 김재리와 대화

 * 시작의 장소
 해당 지역은 차원전쟁이 벌어지기 이전, 알파퀸이 거주하던 지역이다. [아파트 단지터]의 차원종을 섬멸하고 그녀를 찾아보자.
  * 임무: 김재리와 대화 → [싱글:1인] 아파트 단지터 [섬멸 (HARD)] 1회 클리어 → 앨리스와 대화

 * 후배의 봉사
 차원종들은 단독 행동 중인 알파퀸을 계속해서 노리고 있다. [아파트 단지터]에 나타난 차원종들을 섬멸하자.
  * 임무: 앨리스와 대화 → 아파트 단지터 [섬멸 (HARD)] 1회 클리어 → 오세린과 대화

= EP10. 강변과 대교 =
 * 사신의 후계
 [한강 강변길]에서 차원종들의 강력한 지휘관급 개체가 포착되었다. 신중하게 상대하도록 하자.
  * 임무: 오세린과 대화 → [싱글:1인] 한강 강변길 [정복 (V.HARD)] 1회 클리어 → 앨리스와 대화

 * 선배를 위해
 차원종들이 작정하고 진군을 개시했다. [한강 강변길]과 [성수대교]의 차원종들을 격멸하여 알파퀸을 보호하자.
  * 임무: 앨리스와 대화 → 한강 강변길 [정복 (V.HARD)], 성수대교 [정복 (V.HARD)] 각각 1회 클리어 → 김도윤과 대화

 * 아파트 순찰
 적들의 진군이 끝이 없다. [아파트 단지터]의 차원종들을 계속해서 제거하자.
  * 임무: 김도윤과 대화 → 아파트 단지터 [정복 (V.HARD)] 1회 클리어 → 채민우와 대화 

 * 일망타진
 차원종들을 한 장소로 유인해 일제히 격퇴하기로 결정했다. [성수대교]에 출몰한 모든 차원종들을 단숨에 몰아붙이자.
  * 임무: 채민우와 대화 → 성수대교 [정복 (V.HARD)] 2회 클리어 → 김재리와 대화

== 유니온의 완장 ==
 * 오세린의 기념품
 [한강 강변길]에 나타난 식물형 차원종들을 처치하자.
  * 임무: 오세린과 대화 → 한강 강변길 [정복 (V.HARD)] - 맨드란 시즈 20개체 처치 → 오세린과 대화

== 무기 관리 ==
 [한강 강변길]과 [성수대교]에서 무기를 탈취하는 차원종들을 처치하자.
 * 임무: 채민우와 대화 → 한강 강변길 [정복 (V.HARD)] - 맨드레이크 1개체, 성수대교 [정복 (V.HARD)] - 가시 패러사이트 10개체 처치 → 채민우와 대화

= EP11. 갈등의 영웅 =
 * 아파트 방어
 차원종들이 알파퀸에게 위해를 가하게 둘 순 없다. [아파트 단지터]의 차원종들을 몰살하자.
  * 임무: 김재리와 대화 → 아파트 단지터 [정복 (V.HARD)] 2회 클리어 → 채민우와 대화

 * 주둔지 방어
 차원종들이 [한강 강변길]의 특경대 임시 주둔지를 노리고 있다. 신속히 출동하여 해당 지역을 방어하자.
  * 임무: 채민우와 대화 → 한강 강변길 [정복 (V.HARD)] 2회 클리어 → 오세린과 대화

 * 절반의 답
 애쉬와 더스트가 알파퀸을 노리고서 [아파트 단지터]로 진입했다. 그녀와 협력하여 애쉬와 더스트를 몰아내자.
  * 임무: 오세린과 대화 → [싱글:1인] 아파트 단지터 [정복 (V.HARD)] 1회 클리어 → 앨리스와 대화

 * 신강 고등학교로
 알파퀸을 따라 신강고로 지역 이동하여 앨리스에게 상부의 의사를 듣고, 김도윤에게 어떤 지역인지 설명을 요구하자.
  * 임무: 앨리스와 대화 → '''신강고등학교로 이동''' → 앨리스와 대화 → 김도윤과 대화

== 아파트 붕괴 저지 ==

 * 임무: 김도윤과 대화 → 아파트 단지터 [정복 (V.HARD)] - 성장한 맨드란 50개체 처치 → 김도윤과 대화

= EP12. 신강고 =
 * 약속 불이행

  * 임무: 김재리와 대화 → [싱글:1인] 신강고등학교 본관 [침투 (NORMAL)] 1회 클리어 → 채민우와 대화

 * 학교 수호
 교내에 머물러 있는 민간인들의 대피가 이루어질 때까지, [신강고 본관]의 차원종들을 섬멸하자.
  * 임무: 채민우와 대화 → 신강고등학교 본관 [섬멸 (HARD)] 2회 클리어 → 앨리스와 대화

= EP13. 동관의 싸움 =
 * 여왕을 찾아서
 알파퀸을 쫓아 [신강고 동관]을 향해 이동 중인 차원종들을 견제하고, 그녀의 정확한 위치를 파악하기 위해 오세린에게 협조를 구하자.
  * 임무: 앨리스와 대화 → 신강고등학교 동관 [침투 (NORMAL)] 1회 클리어 → 오세린과 대화

 * 복도의 진군
 알파퀸이 교내의 좁은 지역으로 몰릴 것이 우려된다. [신강고 동관]의 차원종들을 섬멸하여 그녀를 보호하자.
  * 임무: 오세린과 대화 → [싱글:1인] 신강고등학교 동관 [섬멸 (HARD)] 1회 클리어 → 김도윤과 대화

 * 그림자
 알파퀸은 [신강고 동관]의 깊은 곳에서 대기 중이다. 부상을 입었을 지 모를 그녀를 찾아 신속히 보호에 나서자.
  * 임무: 김도윤과 대화 → [싱글:1인] 신강고등학교 동관 [섬멸 (HARD)] 1회 클리어 → 김재리와 대화

= EP14. 클론 클로저 =
 * 재리의 분노
 알파퀸 서지수의 클론에 대해 설명을 요구해야 한다. [신강고 본관]과 [신강고 동관]에서 그녀를 상대하고 앨리스에게 사정을 캐묻자.
  * 임무: 김재리와 대화 → 신강고등학교 본관 [정복 (V.HARD)], 신강고등학교 동관 [정복 (V.HARD)] 각각 1회 클리어 → 앨리스와 대화

 * 떨어진 날개
 유니온의 납득할 수 없는 처사를 잠시 뒤로 하고, [신강고 본관]의 차원종들을 격퇴하자. 철저하게.
  * 임무: 앨리스와 대화 → [싱글:1인] 신강고등학교 본관 [정복 (V.HARD)] 1회 클리어 → 채민우와 대화

 * 특경대의 뜻
 [신강고 본관]과 [신강고 동관]에 남은 차원종들을 제거한 다음, 앨리스에게 상부의 방침을 듣도록 하자.
  * 임무: 채민우와 대화 → 신강고등학교 본관 [정복 (V.HARD)], 신강고등학교 동관 [정복 (V.HARD)] 각각 1회 클리어 → 앨리스와 대화

 * 임무: 채민우와 대화 → 신강고등학교 본관 [정복 (V.HARD)] - 프리시온 우로보로스 1개체, 신강고등학교 동관 [정복 (V.HARD)] - 판타노 우로보로스 20개체 처치 → 채민우와 대화
