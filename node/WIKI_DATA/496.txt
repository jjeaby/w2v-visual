'''플레이타임: 약 60분'''(18라운드 6인 플레이 기준)

== 개요 ==
1. 임무확인 : 적(컴퓨터) 유닛 구성이 제시된다.
2. 유닛훈련 : 적 유닛을 모두 제거하기 위한 유닛을 각자 훈련한다.
3. 임무수행 : 싸운다. 이때 '''가장 저렴한 유닛으로 구성된 조합을 가진 플레이어부터 턴이 돌아간다.''' ~~웃긴 조합이 나오는걸 보는것도 묘미다~~ 만약 누군가 임무수행에 성공하면(=모든 적 유닛을 죽이면) 그보다 비싼 조합을 준비한 플레이어들의 턴은 모두 생략된다. 즉 최저가 조합으로 모든 적 유닛을 제거하는데 성공한 1명의 플레이어만이 점수를 획득한다. 


== 진행 방식 ==
=== 게임 설정 투표 ===
처음 시작때 투표를 통해 게임의 길이(총 라운드의 수), 게임 모드, 보급고의 체력을 정한다. 선택할 수 있는 옵션은 다음과 같다.


==== 라운드 ====
9, 12, 15, 18라운드 중 한가지를 선택한다.
18라운드 선택시 6인 플레이 기준으로 약 60분이 소요된다.


==== 게임모드 ====
표준모드 : 말그대로 표준 모드.
유닛제한 : '''[[땅거미 지뢰]], [[파수기]], [[맹독충]]'''을 생산할 수 없다.
프로모드 : 이전에 있던 아케이드모드가 대체된것. 적 유닛 구성을 화면으로만 보여주고 정확한 숫자는 알려주지 않는다. [[전장의 안개]]가 적용되고 지도 전역에 [[점막#s-2.1.2|점막]]이 깔리며, 유닛 훈련에 주어지는 시간이 30초로 줄어든다.


==== 보급고 ====
이 유닛이 파괴되면 해당 라운드에서 바로 패배하게 된다.

기본 보급고 : 체력 400
[[살아남아라! 개복치|개복치]] 보급고 : '''체력 50''' (...)[* 불멸자에게 한대만 맞아도 터진다.(...)]


=== 라운드 진행 ===
==== 적 유닛 확인 ====
각 라운드가 시작되면 12가지 지도와 120가지 적 패턴 중 한가지가 무작위로 선택되어 화면에 보여진다. 등장하는 적의 유형은 거의 항상 '''[ [[저그]] -> [[프로토스]] -> [[테란]] -> 혼합 ]''' 순이다.


==== 유닛 훈련 ====
적 유닛 구성을 확인하고 나면 각자 유닛 훈련을 시작한다. 모드에 따라 다르지만 기본적으로 60초의 제한시간이 주어진다. 유닛을 생산할 때 마다 화면의 오른쪽에 현재까지 뽑은 유닛들의 총 비용이 표시된다. 비용이 적게 들수록 다른사람보다 먼저 턴을 잡을 확률이 오르는 것이기 때문에 알아서 슬기를 발휘하자.
게임이 약간 진행되다 보면 무작위 종족이 금지종족으로 정해지고 해당 종족 유닛은 훈련 할 수 없게 된다. 최대 2가지 종족이 금지된다. 
모든 스킬/업그레이드는 미리 적용되어 있다. 마나는 각자 턴 시작때 각 유닛이 가진 스킬중 가장 마나 소모량이 많은 스킬을 기준으로 초기화 된다.


==== 전투 시작 ====
유닛 훈련 시간이 끝나거나 모든 플레이어가 준비완료 버튼을 누르면 전투가 시작된다. 전투가 시작되기에 앞서, 각 플레이어가 뽑은 유닛의 비용 합계가 [[오름차순]]으로 표시된다. 가장 적은 비용이 든 플레이어부터 한명씩 전투를 진행하게 되는데, 누군가 적 유닛을 모두 제거하는데 성공한다면 그 플레이어보다 비싸게 뽑은 플레이어들은 자동 탈락이다.

예)
p1: 3000원
p2: 2000원
p3: 1000원
이라고 할때, p3부터 라운드에 도전하게 된다. 만약 p3이 적 유닛을 모두 잡는데 성공한다면 p3이 점수를 획득하고 바로 다음 라운드로 넘어간다.
만약 p3이 실패한다면 그 다음으로 싼 p2의 턴이 시작된다. (물론 적 유닛은 초기화된다)

적 유닛들은 플레이어의 보급고위치를 향해 공격해오는데, 스킬은 쓰지 않는다. ([[어택땅]])
제한시간은 60초가 주어진다. 60초가 다 지나도록 적 유닛이 남아있거나, 자신의 [[보급고]]가 파괴되면 다음 사람의 턴으로 넘어간다.
약 40초 지났을때부터 (20초 남았을때부터) 남은 적 유닛중 무작위 1기가 강조 표시된다. 강조 표시된 유닛은 무조건 [[보급고]]를 공격한다. 강조 표시된 유닛을 처치할 경우 또 다른 유닛이 무작위로 선택되어 강조 표시된다.


==== 베팅 ====
다른 플레이어들의 차례때 베팅을 할 수 있다. 해당 플레이어가 라운드에서 승리할 것 같으면 '승'에, 아니면 '패'에 베팅할 수 있다. 1회당 최대 2점까지 걸 수 있으며 만약 자신의 점수가 2점도 없다면 베팅 할 수 없다.(...) 배당률은 보통 2배인데, 한쪽에 치우치게 된다면 반대쪽의 배당률이 오르게 되어있다. 최대 5배까지 오른다.


=== 점수의 획득 ===
처음 게임 시작시 10점에서 시작한다.
각 라운드에서 승리할 경우 10점을 얻는다. 6라운드 부터는 15점, 11라운드 부터 20점, 16라운드 부터 25점을 얻는다.
빠르게 성공할수록 보너스 점수를 얻는다. (남은 시간의 10분의 1만큼 준다)
